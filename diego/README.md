# Teste Vaga Frontend - Me Salva

Um teste com tecnologias frontend para a uma vaga na empresa Me Salva.

## Tecnologias usadas

    - ReactJs
    - Styled Components
    - FlexBox
    - Axios
    - React Router DOM
    - DayJs
    - React input mask

## Rodando localmente

Na pasta raiz rodar:
`npm install`

É preciso ter o create-react-app instalado:
`npm start`

## Demonstração

[Visitar aqui](https://teste-mesalva.web.app)
